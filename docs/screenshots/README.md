# Screenshot Index

These screenshots were captured from the local app after running:

```bash
cd backend
uv run reset-demo --with-comparison
```

They are intended for the written report and final presentation.

## Files

- `01-login.png`: login screen.
- `02-dashboard.png`: dashboard with verdict counts and local-vs-VirusTotal comparison.
- `03-new-scan.png`: manual URL scan form.
- `04-safe-report.png`: safe report example.
- `05-dangerous-report.png`: dangerous fake-domain report example.
- `06-reports-list.png`: stored reports list.
- `07-model-page.png`: model metrics and comparison page.

## Suggested Report Usage

Use these as figure references:

- Dashboard overview: `02-dashboard.png`
- Verdict explanation: `04-safe-report.png` or `05-dangerous-report.png`
- Model evaluation: `07-model-page.png`

The report should still explain that VirusTotal is an external reference, not
absolute truth.
